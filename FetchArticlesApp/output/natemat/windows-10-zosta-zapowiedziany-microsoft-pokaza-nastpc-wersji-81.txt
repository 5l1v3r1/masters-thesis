Następcą systemu Windows 8.1 nie będzie Windows 9. Kolejny system operacyjny Microsoftu będzie nosił nazwę Windows 10 i trafi na rynek w połowie przyszłego roku. Podczas wstępnej prezentacji Windowsa 10 zaznaczono, że nowy system będzie łączył najlepsze elementy Windowsa 7 i Windowsa 8.1. Podczas prezentacji nowego Windowsa, która odbyła się w San Francisco, zapowiedziano, że dziesiątka będzie posiadać menu start. Klasyczny wygląd menu start zostanie wzbogacony o aktywne kafelki, które użytkownik będzie mógł dodawać/usuwać zgodnie z własnymi potrzebami. Menu będzie również wyposażone w znaną z Windowsa 8.1 wyszukiwarkę systemową, która pozwoli na przeszukiwanie zarówno zasobów komputera, jak i internetu. Windows 10 będzie posiadał wysuwane z prawej strony ekranu menu (znane z Windowsa 8.1) oraz – podobnie jak OS X czy Linux – obsługiwał wirtualne pulpity, w których można będzie uruchamiać różne aplikacje. Z prawej strony ekranu będzie się także wysuwać menadżer zadań. Microsoft zapowiedział także, że poprawi wiersz poleceń, do którego będzie można wklejać poszczególne komendy. Kolejne informacje dotyczące Windowsa 10 zostaną zaprezentowane w kwietniu 2015 roku. W środę Microsoft udostępni na swoich serwerach Windows 10 Enterprise Technical Preview, czyli wczesną, pozbawioną wielu funkcji wersje nowego systemu operacyjnego.,Tomasz BaliszewskiNastępcą systemu Windows 8.1 nie będzie Windows 9.

Wielu mniej zaawansowanych użytkowników zapewne przeraża instalacja systemu operacyjnego oraz przygotowania odpowiedniego nośnika do tego zabiegu. Sprawa nie wygląda zazwyczaj tak prosto, jakby mogło się wydawać, gdyż poza technicznymi niuansami nie ma możliwości bezpośredniego pobrania systemu Windows ze strony producenta. Co więcej, nie każdy klient posiada jeszcze nośniki, które otrzymał przy zakupie. Microsoft przygotował całkiem przyjemne i proste rozwiązanie dla osób, które nie chcą stosować pirackich wersji, czy wydawać fortuny na usługi informatyczne. Mówimy tutaj o narzędziu pozwalającym na utworzenie nośnika systemu Windows w szybki i bezproblemowy sposób. W tym krótkim poradniku pokażemy jak krok po kroku stworzyć taki nośnik. Autor: Magdalena Adamowicz Tworzenie nośnika instalacyjnego dla Windows 10 - Poradnik Chcąc zainstalować, bądź ponownie przejść proces instalacji systemu Windows 8.1 istnieje możliwość skorzystania ze specjalnego narzędzia przygotowanego przez firmę Microsoft, które w znacznym stopniu pozwala zaoszczędzić czas i pieniądze. Należy pamiętać, że poniższa instrukcja dotyczy jedynie systemu Windows 8.1. Utworzenie własnego nośnika instalacyjnego może być wykonane za pomocą dysku flash USB lub też płyty DVD. Przed rozpoczęciem działania należy przygotować: Niezawodne połączenie internetowe - jest to szczególnie ważne, gdyż do pobrania jest około 4 GB danych. Należy zwrócić uwagę, że czas pobierania jest uzależniony głównie od szybkości połączenia internetowego. Dysk flash USB - musi posiadać co najmniej 4 GB wolnego miejsca. W związku z tym, że cała zawartość dysku będzie wyczyszczona, zaleca się użycia pustego dysku flash USB albo... Dysk DVD - w tym przypadku wymagane jest posiadanie nagrywarki dysków DVD wbudowanej lub podłączonej do komputera, na którym zamierzasz utworzyć nośnik oraz tym, gdzie będzie instalowany system Windows 8.1. Cała zawartość płyty (o ile jest to możliwe) będzie wyczyszczona, a więc zaleca się użycia pustego dysku DVD. Przed przystąpieniem do działania należy również sprawdzić na komputerze, na którym ma zostać zainstalowany system Windows 8.1 kilka informacji: Procesor 64- lub 32-bitowy - należy pobrać wersję systemu Windows 8.1 (64- lub 32-bitową) odpowiadającą procesorowi w komputerze. Aby sprawdzić, jaki procesor znajduje się w komputerze, przejdź do pozycji Prawy boczny pasek → Ustawienia → Informacje o komputerze lub do pozycji PPM → Menu Start → System i poszukaj wiersza Typ systemu. Wymagania systemowe - zanim zainstalujesz system Windows 8.1, należy zapoznać się z wymaganiami systemowymi. Warto także odwiedzić stronę internetową producenta komputera, aby uzyskać informacje na temat zaktualizowanych sterowników i zgodności sprzętu. Język systemu Windows - podczas instalacji systemu trzeba wybrać ten sam język. Aby sprawdzić, jaki język jest obecnie używany, przejdź do PPM → Menu Start → Panel sterowania → Język lub Prawy boczny pasek → Ustawienia → Zmień ustawienia komputera → Czas i język → Region i język. Wersja systemu Windows - należy także wybrać tę samą wersję systemu Windows. Może to być wersja Windows 8.1, Windows 8.1 N, Windows 8.1 Pro lub Windows 8.1 Pro N. Aby sprawdzić, jaka wersja jest zainstalowana na komputerze, przejdź do pozycji Prawy boczny pasek → Ustawienia → Informacje o komputerze lub do pozycji PPM → Menu Start → System i poszukaj wiersza Wersja systemu Windows. Czym różnią się wersje systemu Winowds 8.1? Windows 8.1 N i Windows 8.1 Pro N - zawierają te same funkcje, co odpowiednio Windows 8.1 i Windows 8.1 Pro, oprócz obsługi pewnych technologii multimedialnych (Windows Media Player, Muzyka, Wideo, Skype). W związku z tym, aby móc odtwarzać i tworzyć dyski CD audio, dyski DVD wideo oraz pliki multimedialne, przesyłać strumieniowo utwory muzyczne, robić i przechowywać zdjęcia oraz korzystać z kamery internetowej, należy zaopatrzyć się w odpowiednie programy firmy Microsoft lub innych firm. Kiedy wszystko będzie gotowe, podłącz dysk flash USB lub włóż dysk DVD oraz kliknij w poniższy link. Po pobraniu kreatora do tworzenia nośnika, uruchom go i postępuj zgodnie z instrukcjami kreatora. Pobierz narzędzie do tworzenia nośnika dysku systemu Windows 8.1 Proces tworzenia nośnika Krok 1: Należy wybrać język systemu spośród 37 propozycji, odpowiadający językowi naszemu poprzedniego systemu. Następnym etapem jest określenie wersji systemu, który chcielibyśmy zainstalować (Windows 8.1, Windows 8.1 N, Windows 8.1 Pro, Windows 8.1 Pro N). Musimy również ustalić rodzaj architektury, spośród 32-bitowej (x86) lub 64-bitowej (x64). Po uzupełnieniu pól, należy wybrać przycisk Dalej, by kontynuować. Krok 2: W kolejnym etapie należy wybrać na jakim rodzaju nośnika chcesz zapisać plik instalacyjny. Wybierając dysk flash USB, należy pamiętać, by wybrać pusty, gdyż zostanie on wcześniej całkowicie wyczyszczony oraz aby ilość wolnego na nim miejsca nie była mniejsza od 4 GB. Wybierając dysk DVD, należy pamiętać, że do tego celu wymagana jest nagrywarka DVD. Plik instalacyjny zostanie zapisany w formacie ISO, który następnie trzeba nagrać na dany nośnik. W tym przypadku dysk powinien posiadać przynajmniej 4 GB wolnego miejsca. Po wybraniu opcji należy wybrać przycisk Dalej, by kontynuować. Krok 3: Jest to ostatni moment by przenieść ważne pliki z dysku flash USB, gdyż za chwilę zostanie on całkowicie wyczyszczony w celu przygotowania go do instalacji. Po wybraniu odpowiedniego dysku należy kliknąć przycisk Dalej, by kontynuować. Wybierając drugą opcje, będzie trzeba wybrać miejsce docelowe, w którym ma zostać zapisany pobrany system w formacie ISO. Krok 4: Otrzymamy komunikat, który informuje nas o tym, że zawartość nośnika została z powodzeniem wyczyszczona. Wybierz przycisk OK, by kontynuować lub Anuluj, jeśli chcesz zatrzymać proces tworzenia nośnika. Krok 5: Podczas kolejnego etapu następuje pobranie pliku instalacyjnego. W zależności od kondycji łącza internetowego oraz mocy komputera może to trwać od kilku do kilkudziesięciu minut (pobierane jest około 4 GB). Krok 6: Przechodzimy automatycznie do głównego etapu naszego poradnika, jakim jest tworzenie nośnika instalacyjnego (w tym przypadku dysku flash USB). Krok 7: Ostatnim krokiem kreatora nośnika instalacji jest komunikat o zakończeniu procesu tworzenia dysku flash USB. Przed rozpoczęciem instalacji należy upewnić się, czy posiadamy działający klucz systemu Windows 8.1. Na sam koniec wybieramy przycisk Zakończ. Krok 8: Jak łatwo zauważyć, cały proces tworzenia nośnika instalacji jest banalnie prosty i przyjemny. Po zakończeniu działania kreatora zostanie wyświetlony nasz dysk z plikami systemowymi. Po zakończeniu tworzenia nośnika instalacyjnego należy wykonać jedną z następujących czynności, aby zainstalować system Windows 8.1: Jeśli chcesz zainstalować system na już istniejącym, podłącz nośnik instalacyjny do komputera, na którym chcesz zainstalować system Windows 8.1, przejdź do niego w Eksploratorze plików, otwórz plik setup.exe, a następnie postępuj zgodnie z instrukcjami dotyczącymi instalacji. Jeśli komputer skonfigurowano do rozruchu z dysku USB, podłącz dysk flash USB do komputera, na którym chcesz zainstalować system Windows 8.1, uruchom go ponownie, wybierz pozycję Zainstaluj teraz, a następnie postępuj zgodnie z instrukcjami dotyczącymi instalacji. Jeśli komputer skonfigurowano do rozruchu z dysku DVD, włóż dysk DVD do komputera, uruchom go ponowne, wybierz pozycję Zainstaluj teraz, a następnie postępuj zgodnie z instrukcjami dotyczącymi instalacji. Jeśli chcesz zainstalować system Windows 8.1 na innej partycji bądź na czystym dysku, należy włożyć dysk DVD lub flash USB oraz ponownie uruchomić komputer. Pojawi się komunikat „Naciśnij dowolny klawisz, aby wykonać rozruch z dysku DVD”, naciśnij klawisz. Jeśli komunikat nie zostanie wyświetlony lub wciąż uruchamia się bieżąca wersja systemu Windows, być może trzeba zmienić kolejność rozruchu w ustawieniach systemu BIOS na komputerze. W przypadku, gdy uda nam się z powodzeniem uruchomić nośnik z instalacją systemu, wybierz pozycję Zainstaluj teraz, a następnie postępuj zgodnie z instrukcjami dotyczącymi instalacji. Jak zmienić kolejność rozruchu? Aby zmienić kolejność rozruchu w ustawieniach systemu BIOS na komputerze zazwyczaj bezpośrednio po uruchomieniu komputera trzeba nacisnąć kombinację klawiszy (na przykład F2, F12, Delete lub Esc). Ta kombinacja klawiszy jest różna dla różnych komputerów, dlatego być może musisz odwiedzić witrynę producenta płyty głównej, aby uzyskać informacje na temat odpowiedniej kombinacji klawiszy dla swojego komputera. Jeśli po ponownym uruchomieniu komputera i naciśnięciu wymaganej kombinacji klawiszy wciąż uruchamia się bieżąca wersja systemu Windows poczekaj, aż pojawi się prośba o zalogowanie do systemu Windows, a następnie ponownie uruchom komputer i spróbuj ponownie.

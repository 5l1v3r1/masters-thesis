Pobór mocy OC - spoczynek i obciążenie 42 - Podsumowanie - Drogie cztery wątki Odkąd zadebiutował Intel Pentium G4560, będący w przeciwieństwie do poprzedników procesorem dwurdzeniowym z technologią Hyper-Threading, czyli obsługującym cztery wątki jednocześnie, pozycja rodziny Intel Core i3 wyraźnie osłabła. Mimo wszystko Intel nie zmienił hierarchii czy globalnej specyfikacji reszty układów, zatem prawdziwe cztery rdzenie znajdziemy dopiero w serii Core i5. Gdzie w takim razie miejsce dla procesorów pokroju Intel Core i3-7100, skoro wydajnościowa przewaga względem Pentium G4560 okazuje się stosunkowo niewielka, natomiast różnica cenowa jest blisko dwukrotna? Tutaj dokonało się zadziwiające przetasowanie, wywracające dotychczasowy porządek rzeczy, nawet pomimo odebrania Pentium G4560 dodatkowych instrukcji. Jak wypadnie Intel Core i3-7100 dysponujący pełnym potencjałem architektury Kaby Lake, dowiecie się czytając poniższą publikację. Autor: Sebastian Oktaba Jeszcze niedawno w świecie procesorów wszystko było zdecydowanie prostsze - Intel działał według klarownej strategii TICK-TOCK, gdzie pierwszy krok odpowiadał za wdrożenie niższego procesu technologicznego, natomiast drugi przynosił nową architekturę. Starczy tutaj wspomnieć chociażby generacje Sandy/Ivy Bridge albo Haswel/Broadwell. Jednak opracowywanie kolejnych litografii w regularnych odstępach czasu okazało się problematyczne oraz ekonomicznie nieopłacalne, więc producent musiał zmienić podejście wprowadzając dodatkowe ogniwo do powyższego łańcucha - optymalizację. Namacalnym przykładem zastosowania mechanizmu Process-Architecture-Optimization jest właśnie Kaby Lake, bowiem pierwszymi 14 nm desktopowymi układami Intela były Broadwelle, potem przyszła nowa architektura Skylake, natomiast Kaby Lake stanowi kolejne i niekonieczne ostatnie usprawnienie tejże litografii. Dlatego siódma generacja przynosi raczej symboliczne zmiany pod względem czystej wydajności zegar w zegar względem poprzedników, niemniej sporo dodatków mających zwiększyć ogólną funkcjonalność platformy, jak również przetasowania w hierarchii układów. Gdzie jest miejsce dla procesorów pokroju Intel Core i3-7100, skoro wydajnościowa przewaga względem Pentium G4560 okazuje się stosunkowo niewielka, natomiast różnica cenowa jest blisko dwukrotna? Intel Core i3-7100 jest procesorem dwurdzeniowym obsługującym technologię Hyper-Threading, zatem wykonującym do czterech wątków jednocześnie, dzięki dodatkowym rdzeniom logicznym. Podobne rozwiązanie od niepamiętnych czasów funkcjonuje w jednostkach Core i7, będąc zarazem ich główną przewagą względem czystych czterordzeniowych Intel Core i5. Jeszcze do niedawna identyczna zależność zachodziła między seriami Core i3 oraz Pentium. Obsługa Hyper-Threadingu zwykle owocuje solidnym przyrostem wydajności, zwłaszcza w aplikacjach wielowątkowych, jak również wymagających grach komputerowych. Ponieważ coraz częściej do uruchomienia nowych tytułów dwurdzeniówki nie wystarczają, procesor czterowątkowy wydaje się dzisiaj absolutnym minimum, jeżeli poważnie myślimy o cyfrowej rozrywce. Intel Core i3-7100 spełnia minimalne wymagania wszystkich współczesnych produkcji, aczkolwiek Hyper-Threading nie zastąpi prawdziwych fizycznych rdzeni, będących jedynie całkiem efektywnym półśrodkiem. Dobra, spójrzmy na szczegółową specyfikację techniczną omawianego Intel Core i3-7100. Intel Core i3-7100 jest oficjalnym następcą Intel Core i3-6100, pracującym z bazową częstotliwością 3900 MHz, czyli dokładnie 200 MHz wyższą od poprzednika. Różnica w wydajności będzie zatem odpowiadać różnicy w taktowaniu. Zegary pod obciążeniem pozostają niezmienne, bowiem rodzina Core i3 wsparcia technologii Turbo Boost ciągle nie otrzymała. Mnożnik jest oczywiście zablokowany - amatorom podkręcania Intel dedykował Core i3-7350K, będący obecnie najsłabszym procesorem do overclockingu (niestety, również kompletnie nieopłacalnym). Opisywany model jak wszystkie Kaby Lake posiada wbudowany dwukanałowy kontroler pamięci DDR3-1600L oraz DDR4-2400. Opisywana jednostka obsługuje też zestaw instrukcji AVX, F16C i FMA3, których Pentium G4560 pożałowano. Rozszerzenia zbytnio nie podnoszą wydajności, dlatego potencjalny użytkownik zyskuje niewiele. Ostatnim istotnym elementem procesora jest zintegrowany układ graficzny Intel HD 630, posiadający między innymi sprzętowe dekodery HEVC/VP9, które docenią zawłaszcza maniacy filmowych oraz YouTube. Namacalnym przykładem zastosowania mechanizmu Process-Architecture-Optimization jest właśnie Kaby Lake, bowiem pierwszymi 14 nm desktopowymi układami Intela były Broadwelle, potem przyszła nowa architektura Skylake, natomiast Kaby Lake stanowi kolejne i niekonieczne ostatnie usprawnienie tejże litografii. Dlatego siódma generacja przynosi raczej symboliczne zmiany pod względem czystej wydajności zegar w zegar względem poprzedników, niemniej sporo dodatków mających zwiększyć ogólną funkcjonalność platformy, jak również przetasowania w hierarchii układów. Gdzie jest miejsce dla procesorów pokroju Intel Core i3-7100, skoro wydajnościowa przewaga względem Pentium G4560 okazuje się stosunkowo niewielka, natomiast różnica cenowa jest blisko dwukrotna? Intel Core i3-7100 jest procesorem dwurdzeniowym obsługującym technologię Hyper-Threading, zatem wykonującym do czterech wątków jednocześnie, dzięki dodatkowym rdzeniom logicznym. Podobne rozwiązanie od niepamiętnych czasów funkcjonuje w jednostkach Core i7, będąc zarazem ich główną przewagą względem czystych czterordzeniowych Intel Core i5. Jeszcze do niedawna identyczna zależność zachodziła między seriami Core i3 oraz Pentium. Obsługa Hyper-Threadingu zwykle owocuje solidnym przyrostem wydajności, zwłaszcza w aplikacjach wielowątkowych, jak również wymagających grach komputerowych. Ponieważ coraz częściej do uruchomienia nowych tytułów dwurdzeniówki nie wystarczają, procesor czterowątkowy wydaje się dzisiaj absolutnym minimum, jeżeli poważnie myślimy o cyfrowej rozrywce. Intel Core i3-7100 spełnia minimalne wymagania wszystkich współczesnych produkcji, aczkolwiek Hyper-Threading nie zastąpi prawdziwych fizycznych rdzeni, będących jedynie całkiem efektywnym półśrodkiem. Dobra, spójrzmy na szczegółową specyfikację techniczną omawianego Intel Core i3-7100. Intel Core i3-7100 jest oficjalnym następcą Intel Core i3-6100, pracującym z bazową częstotliwością 3900 MHz, czyli dokładnie 200 MHz wyższą od poprzednika. Różnica w wydajności będzie zatem odpowiadać różnicy w taktowaniu. Zegary pod obciążeniem pozostają niezmienne, bowiem rodzina Core i3 wsparcia technologii Turbo Boost ciągle nie otrzymała. Mnożnik jest oczywiście zablokowany - amatorom podkręcania Intel dedykował Core i3-7350K, będący obecnie najsłabszym procesorem do overclockingu (niestety, również kompletnie nieopłacalnym). Opisywany model jak wszystkie Kaby Lake posiada wbudowany dwukanałowy kontroler pamięci DDR3-1600L oraz DDR4-2400. Opisywana jednostka obsługuje też zestaw instrukcji AVX, F16C i FMA3, których Pentium G4560 pożałowano. Rozszerzenia zbytnio nie podnoszą wydajności, dlatego potencjalny użytkownik zyskuje niewiele. Ostatnim istotnym elementem procesora jest zintegrowany układ graficzny Intel HD 630, posiadający między innymi sprzętowe dekodery HEVC/VP9, które docenią zawłaszcza maniacy filmowych oraz YouTube.

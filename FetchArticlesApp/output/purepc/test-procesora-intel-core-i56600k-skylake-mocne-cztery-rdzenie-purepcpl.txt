Entuzjaści długo czekali na premierę Socket LGA 1151 dedykowanego procesorom Intel Skylake, który wprowadza nową architekturę i szereg fundamentalnych zmian, mogących wymiernie wpłynąć na wydajność całej platformy. Dlatego zapewne ucieszy Was wiadomość, że dzisiaj punktualnie o godzinie 14.00 kurtyna opadła - przedstawienie możemy zaczynać! Oprócz najszybszego reprezentanta nowej rodziny procesorów w postaci Core i7-6700K, naszą redakcję odwiedził również trochę słabszy model Core i5-6600K, potencjalnie jednak bardziej atrakcyjny dla kupujących. Solidne cztery rdzenie, 14 nm proces technologiczny, odblokowany mnożnik, wsparcie dla pamięci DDR4 i DDR3L - wszystko razem skonsolidował Skylake. Pozostaje tylko pytanie jak Intel Core i5-6600K wypada w porównaniu do poprzedników: Core i5-5675C, Core i5-4690K, Core i5-3570K i Core i5-2500K? Autor: Sebastian Oktaba Zacznijmy od informacji absolutnie podstawowej - platforma LGA 1151 jest kompatybilna wyłącznie z procesorami Intel Core szóstej generacji (Skylake), więc kupując nową jednostkę obliczeniową, musimy nabyć również płytę główną z chipsetem Intel Z170/H170. Skylake nie będzie pasował do konstrukcji z układami logiki Intel Z97/H97/Z87 itp. Ponieważ kontroler FIVR przeniesiono na płytę główną, rezygnując z pomysłu jego zintegrowania uparcie forsowanego w Haswellach, kwestie zarządzania zasilaniem uległy zasadniczej zmianie. Mimo wszystko współczynnik TDP testowanego Skylake wynosi nieskromne 91W okazując się wyższy od Broadwelli (65W) oraz Devil's Canyon (88W). Jak zostało powiedziane, Core i5-6600K wykonano w świeżutkim 14 nm procesie technologicznym, któremu ścieżki na desktopowej arenie przecierały Broadwelle, zaprezentowane zaledwie na początku czerwca bieżącego roku. Intel zazwyczaj wprowadza dwie generacje procesorów w obrębie jednego gniazda, zatem oprócz Skylake posiadacze nowych płyt głównych Intel Z170/H170 prawdopodobnie ugoszczą również jego następcę, aczkolwiek przypadek Broadwella jasno pokazuje, że niczego nie można być stuprocentowo pewnym (brak wsparcia dla chipsetów Intel Z87/H87/B85/H81). Czy Intel Core i5-6600K godnie zastąpi na stanowisku swoich poprzedników i będzie dla Socket LGA 1151 tym, czym dla Socket LGA 1150 był Core i5-2500K? Intel Core i5-6600K jest czterordzeniowym procesorem, który w przeciwieństwie do droższego Core i7-6700K nie wspiera technologii Hyper-Threading, więc obsługuje jednocześnie cztery wątki zamiast ośmiu. Użyteczność tego rozwiązania jest odczuwalna głównie w wyspecjalizowanych programach (m.in.: rendering, szyfrowanie), rzadziej w grach komputerowych, chociaż nie brakuje produkcji wyraźnie zyskujących przy aktywnym Hyper-Threadingu m.in.: Crysis 3, Battlefield 4 oraz Watch Dogs. Bazowe taktowanie Intel Core i5-6600K wynosi 3500 MHz, automatycznie wzrastając w trybie Turbo Boost do maksymalnej wartości 3900 MHz. Intel Core i5-4690K pracuje z identycznymi częstotliwościami, dzięki czemu łatwiej będzie porównać wydajność zegar w zegar obydwu procesorów - zwłaszcza wpływ samej architektury. Litera „K” umieszczona w nazwie oznacza odblokowany mnożnik, pozwalający na swobodne podkręcanie (maksymalna wartość wynosi x83), zaś chipset Intel Z170 umożliwia jeszcze regulowanie szyny BCLK odseparowanej od magistrali DMI/PEG. Miejmy tylko nadzieję, że Skylake będzie bardziej elastyczny niż Broadwell, jakiego nawet radykalne podnoszenie napięcia nie zmotywowało do przekroczenia magicznej granicy 4200-4300 MHz. Intel Core i5-6600K otrzymał 6MB pamięci podręcznej trzeciego poziomu (L3), aczkolwiek po wydzielonym module eDRAM wciśniętym do Broadwelli zostały tylko wspomnienia, przynajmniej jeśli mówimy o wersjach desktopowych Skylake-S. Podobnie jak Intel Core i7-6700K, również Core i5-6600K wyposażono w kontroler pamięci DDR3L-1600 oraz DDR4-2133, pozwalający użytkownikowi wybierać między modułami obydwu standardów. Jednak w praktyce większość producentów płyt głównych z chipsetami Intel Z170 zainstalowała w swoich produktach wyłącznie gniazda DDR4, bowiem niskonapięciowe DDR3L (1.35V) stanowią margines rynku. Opisywany procesor posiada także 16 linii PCI-Express 3.0, wsparcie licznych instrukcji oraz zintegrowany układ graficzny Intel HD 530. Kwestii iGPU poświęcona zostanie osobna publikacja, uwzględniająca też Broadwelle, Devil's Canyon i oczywiście APU Kaveri, natomiast dzisiaj dokładnie sprawdzona zostanie wydajność rdzeni x86. Warto jeszcze wspomnieć, że startowa cena Intel Core i5-6600K wynosi 243 dolarów amerykańskich, czyli jakieś 1150 złotych, aczkolwiek spekulacje o słabej dostępności Skylake w pierwszych miesiącach po premierze, mogą służyć podbijaniu ceny detalicznej.

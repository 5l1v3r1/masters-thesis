Windows Server 2008 jest kolejną serwerowa wersją Windows, stworzoną przez firmę Microsoft. Premiera systemu odbędzie się dopiero w przyszłym roku, ale już teraz dostępna jest trzecia wersja beta owego OS-a. Jak dużo się zmieniło od wydanego cztery lata temu Windows Server 2003 i co nowego znajdziemy w serwerowych oknach datowanych na 2008 rok? O tym oczywiście przeczytacie w dalszej części tekstu. Serwerowe edycje Windows zawsze miały je zawyżone. Należy też zwrócić uwagę na jeszcze jeden szczegół. Windows Server 2008 został stworzony na bazie Windows Vista. W wymaganiach można zauważyć opcję Core. Co to takiego? Jest to nowy tryb instalacji, który umożliwia instalację systemu z mniejszą ilością usług bez trybu graficznego. Instalacja Instalator Windows Server 2008 wygląda dokładnie tak samo jak w Windows Vista. Zmienione zostały tylko kolory i tło.

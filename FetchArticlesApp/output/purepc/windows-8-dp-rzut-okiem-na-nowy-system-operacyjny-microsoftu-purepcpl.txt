Właśnie tego jesiennego dnia Microsoft wypuścił całkiem (jak się teraz okazuje) udany system Windows 7, dotychczas sprzedany w ponad 450 milionach kopii. W szybkim czasie „siódemka” zdobyła zaufanie użytkowników, którzy momentalnie zapominali o Viście i częściowo XP. Niemniej ostatnio gigant z Redmond podczas konferencji Build Windows narobił wszystkim apetytu - zaprezentował godnego następcę, który ma przynieść "zieloną" rewolucję. Interfejs Metro, nowe funkcje wizualne oraz użytkowe teoretycznie powinny zachęcić użytkownika do zmiany systemu na Windows 8. Czy nowe "okienka" nawet we wczesnej becie mogą czymś zaskoczyć? Dzięki wydaniu specjalnej wersji deweloperskiej mamy przyjemność przetestowania tego systemu w prawie pełnej krasie. Jak w naszych oczach wypadł Windows Developer Preview? Czego możemy się spodziewać w finalnej wersji? Proponujemy usiąść wygodnie w fotelu i przeżyć z nami (nie)małą rewolucję. Autor: Arkad - Arkadiusz Pracki Dużą burzę wywołała niedawna konferencja Build Windows, na której zaprezentowano działający już system Windows 8 na architekturze x86, jak również ARM. Dowiedzieliśmy się przy okazji, iż system zostanie rozesłany przez ograniczony kanał deweloperski. Wielu zastanawiało się wtedy, ile czasu minie do ewentualnego wycieku nowego "dziecka" giganta z Redmond. Sprytny Microsoft uprzedził jednak wszystkich śliniących się na taką wiadomość. 14 września tuż po zakończeniu konferencji, na oficjalnej stronie Build Windows został udostępniony Windows Developer Build - nie mogliśmy więc przepuścić okazji do przetestowania następcy Windows 7. Nie owijając już w bawełnę - przechodzimy do instalacji. Wymagania Windows Developer Build Procesor o taktowaniu 1 GHz 1 GB pamięci RAM dla wersji 32-bit lub 2 GB dla 64-bit 16 GB wolnego miejsca na dysku dla 32-bit lub 20 GB dla 64-bit Karta graficzna obsługująca DirectX 9 oraz WDDM 1.0 Aplikacje Metro wymagają rozdzielczości co najmniej 1024 x 768 Platforma testowa - Notebook Compaq CQ60 Procesor AMD Athlon X2 1.9 GHz 2 GB pamięci RAM DDR2 Karta graficzna Nvidia GeForce 8200M Rozdzielczość 1366 x 768 Windows Developer Build x86 (32-bit) Po załadowaniu znanego skądinąd instalatora wybieramy język systemu (w tym przypadku tylko angielski), strefę czasową oraz język wprowadzania. Oczywiście istnieje możliwość aktualizacji systemu Windows 7, jednak nie polecamy tej opcji w przypadku wersji deweloperskiej. Instalacja systemu przebiegła wyjątkowo sprawnie - wszystko zajęło około 8 minut przy użyciu pamięci Flash. Takim ekranem wita nas Windows 8, przy czym warto wspomnieć o nowym oknie wyboru, nie pozwalającym na uruchomienie innego systemu niż z rodziny Windows. Pierwszy minus dla samolubnego Microsoftu! Przy ciągłym używaniu systemu Windows 8 jest on już początkowo załadowany. Wybierając Windows 7 komputer musi zostać zrestartowany lub na odwrót (możliwe do skonfigurowania w tym samym oknie).

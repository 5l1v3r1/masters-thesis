Test pamięci DDR4 2133 vs 3200 MHz - Programy #2 44 - Test pamięci DDR4 2133 vs 3200 MHz - Gry komputerowe#1 45 - Test pamięci DDR4 2133 vs 3200 MHz - Gry komputerowe#2 46 - Podsumowanie - Zegarmistrz Światła Purpurowy W cieniu odblokowanych procesorów Intel Core i5-8600K i Core i7-8700K, zadebiutował również zablokowany Core i5-8400, którego początkowo niewielu zainteresowanych 8. generacją Core typowało jako najciekawszy model Coffee Lake. Tymczasem pierwsze testy pokazują, że niepozorny Core i5-8400 może dokonać na rynku totalnego spustoszenia, bowiem w rzeczywistości okazuje się wydajniejszy, niż wskazywałaby papierowa specyfikacja techniczna. Zresztą, mówimy o najtańszej sześciordzeniowej jednostce, zastępującej w portfolio Niebieskich czterordzeniowego Core i5-7400, więc zmiana jakościowa jest ogromna. Amatorów mocnych wrażeń zapraszam na pojedynek Intel Core i5-8400 kontra AMD Ryzen 5 1600, czyli przystępnych cenowo sześciordzeniowych procesorów. Autor: Sebastian Oktaba Architektura Coffee Lake wprowadza niewiele zmian względem poprzedników, będąc trzecią iteracją Skylake, jednak oprócz dodatkowych rdzeni, procesorom zwiększono także rozmiar pamięci podręcznej L1/L2/L3. Wszystkie jednostki wykonano w ulepszonej 14 nm litografii, zapewniającej dobrą efektywność energetyczną, wysokie fabryczne taktowania i perspektywę niezłego overclockingu (jeśli posiadają odblokowany mnożnik). Coffee Lake wciąż korzystają z gniazda LGA 1151, aczkolwiek pomimo fizycznego podobieństwa do poprzedników, wymagają układów logiki z rodziny 300. Powodem zmiany jest zwiększenie liczby pinów VSS i VCC (masa / zasilanie), odróżniających obydwa sockety, dlatego nowości nie współpracują z płytami głównymi zbudowanymi na chipsetach Intel Z170/270 oraz niższymi. Dla równowagi - Kaby Lake nie zadziałają na modelach z układami Intel Z370, więc jakiekolwiek krzyżowanie bliźniaczych platform jest niemożliwe. Co gorsza, tańsze odmiany płyt głównych dedykowanych Coffee Lake mają pojawić się dopiero na początku pierwszego kwartału 2018 roku... Core i5-8400 to aktualnie najtańszy procesor z sześcioma rdzeniami w ofercie Intela, a jednocześnie najtańsze sześć rdzeni na rynku, wyprzedzające pod tym względem nawet dotychczasowego lidera - AMD Ryzen 5 1600. Test procesora Intel Core i5-8600K - Rzeźnik zwany Coffee Lake Cztery rdzenie w mainstreamie Intela panowały od niepamiętnych czasów, ostatnio przynosząc już symboliczne wzrosty wydajności (Core i5-6400 → Core i5-7400), toteż środkowa półka wzbudzała umiarkowane emocje, zwłaszcza odkąd Niebiescy zablokowali tym jednostkom podstawowy zakres overclockingu (m.in.: wymuszenie maksymalnego mnożnika Turbo Boost). Sytuacja zmieniła się jednak diametralnie dzięki AMD Ryzen 5 1600, który za rozsądne pieniądze zaoferował sześć fizycznych rdzeni i dwanaście wątków, przewyższających klasowych konkurentów w większości programów, a niejednokrotnie dorównujących również w grach komputerowych. Ponieważ cztery „jajka” bywają niewystarczające w poważniejszych zastosowaniach, AMD Ryzen 5 1600 błyskawicznie zdobył ogromną popularność, mocno podkopując pozycję zablokowanych Core i5-7000. Niebiescy musieli dokonać zdecydowanego kontrataku, skutkiem czego testowany dzisiaj Core i5-8400 jest modelem sześciordzeniowym. Rynek procesorów stricte konsumenckich po latach stagnacji ruszył z miejsca, dzięki czemu konsumenci nareszcie zyskali wybór. Koniunktura jest lepsza, niż można było oczekiwać. Test procesora Intel Core i7-8700K - Premiera Coffee Lake Intel Core i5-8400 jest procesorem sześciordzeniowym, pozbawionym obsługi technologii Hyper-Threading zarezerwowanej dla modeli Core i7 oraz wybranych Pentiumów, dlatego wykonuje maksymalnie sześć wątków. Jednostce zwiększono natomiast rozmiar pamięci cache L1 (128 → 192 kB), cache L2 (1024 → 1536 kB) oraz cache L3 (6144 → 9216 kB), mającej niebagatelne znaczenie w grach komputerowych. Dwukanałowy kontroler pamięci obsługuje teraz oficjalnie moduły DDR4-2666, chociaż może współpracować ze znacznie szybszymi zestawami (np.: 3200 MHz jakich używam w testach dodatkowych), warunkiem jest jednak posiadanie płyty głównej z chipsetem Intel Z370. Bazowe taktowanie Core i5-8400 wynosi 2800 MHz, niemniej faktyczna wartość jest znacznie wyższa, sięgając 3800 MHz podczas obciążenia wszystkich rdzeni. Maksymalne przyspieszenie Turbo Boost dla pojedynczego wątku to natomiast 4000 MHz. Trzeba przyznać, że taktowania są naprawdę wysokie, stawiając Core i5-8400 tylko nieznacznie za Core i5-8600K, osiągającego przeważnie 4100 MHz. Żeby było ciekawiej - nawet na chłodzeniu fabrycznym, pomimo wysokich temperatur, Core i5-8400 utrzymuje takie częstotliwości. PS: Na płycie głównej jakiej używam, nie udało się wymusić maksymalnego Turbo Boost dla wszystkich rdzeni (4000 MHz). Procesor pomimo ustawiania mnożnika x40 dla każdego rdzenia, synchronizacji oraz Enhanced Turbo, nadal pracował przy 3800 MHz. Problemu nie stanowi jednak obsługa szybszych modułów RAM, bowiem to uzależnione jest wyłącznie od chipsetu płyty głównej, a obecnie dostępne są tylko konstrukcje na Intel Z370. 1 2 3 4 5 6 7 8 9 … następna › ostatnia » Test Intel Core i5-8400 vs AMD Ryzen 5 1600 - Test pamięci DDR4 2133 vs 3200 MHz - Gry komputerowe#2 46 - Podsumowanie - Zegarmistrz Światła Purpurowy W cieniu odblokowanych procesorów Intel Core i5-8600K i Core i7-8700K, zadebiutował również zablokowany Core i5-8400, którego początkowo niewielu zainteresowanych 8. generacją Core typowało jako najciekawszy model Coffee Lake. Tymczasem pierwsze testy pokazują, że niepozorny Core i5-8400 może dokonać na rynku totalnego spustoszenia, bowiem w rzeczywistości okazuje się wydajniejszy, niż wskazywałaby papierowa specyfikacja techniczna. Zresztą, mówimy o najtańszej sześciordzeniowej jednostce, zastępującej w portfolio Niebieskich czterordzeniowego Core i5-7400, więc zmiana jakościowa jest ogromna. Amatorów mocnych wrażeń zapraszam na pojedynek Intel Core i5-8400 kontra AMD Ryzen 5 1600, czyli przystępnych cenowo sześciordzeniowych procesorów. Autor: Sebastian Oktaba Architektura Coffee Lake wprowadza niewiele zmian względem poprzedników, będąc trzecią iteracją Skylake, jednak oprócz dodatkowych rdzeni, procesorom zwiększono także rozmiar pamięci podręcznej L1/L2/L3. Wszystkie jednostki wykonano w ulepszonej 14 nm litografii, zapewniającej dobrą efektywność energetyczną, wysokie fabryczne taktowania i perspektywę niezłego overclockingu (jeśli posiadają odblokowany mnożnik). Coffee Lake wciąż korzystają z gniazda LGA 1151, aczkolwiek pomimo fizycznego podobieństwa do poprzedników, wymagają układów logiki z rodziny 300. Powodem zmiany jest zwiększenie liczby pinów VSS i VCC (masa / zasilanie), odróżniających obydwa sockety, dlatego nowości nie współpracują z płytami głównymi zbudowanymi na chipsetach Intel Z170/270 oraz niższymi. Dla równowagi - Kaby Lake nie zadziałają na modelach z układami Intel Z370, więc jakiekolwiek krzyżowanie bliźniaczych platform jest niemożliwe. Co gorsza, tańsze odmiany płyt głównych dedykowanych Coffee Lake mają pojawić się dopiero na początku pierwszego kwartału 2018 roku... Core i5-8400 to aktualnie najtańszy procesor z sześcioma rdzeniami w ofercie Intela, a jednocześnie najtańsze sześć rdzeni na rynku, wyprzedzające pod tym względem nawet dotychczasowego lidera - AMD Ryzen 5 1600. Test procesora Intel Core i5-8600K - Rzeźnik zwany Coffee Lake Cztery rdzenie w mainstreamie Intela panowały od niepamiętnych czasów, ostatnio przynosząc już symboliczne wzrosty wydajności (Core i5-6400 → Core i5-7400), toteż środkowa półka wzbudzała umiarkowane emocje, zwłaszcza odkąd Niebiescy zablokowali tym jednostkom podstawowy zakres overclockingu (m.in.: wymuszenie maksymalnego mnożnika Turbo Boost). Sytuacja zmieniła się jednak diametralnie dzięki AMD Ryzen 5 1600, który za rozsądne pieniądze zaoferował sześć fizycznych rdzeni i dwanaście wątków, przewyższających klasowych konkurentów w większości programów, a niejednokrotnie dorównujących również w grach komputerowych. Ponieważ cztery „jajka” bywają niewystarczające w poważniejszych zastosowaniach, AMD Ryzen 5 1600 błyskawicznie zdobył ogromną popularność, mocno podkopując pozycję zablokowanych Core i5-7000. Niebiescy musieli dokonać zdecydowanego kontrataku, skutkiem czego testowany dzisiaj Core i5-8400 jest modelem sześciordzeniowym. Rynek procesorów stricte konsumenckich po latach stagnacji ruszył z miejsca, dzięki czemu konsumenci nareszcie zyskali wybór. Koniunktura jest lepsza, niż można było oczekiwać. Test procesora Intel Core i7-8700K - Premiera Coffee Lake Intel Core i5-8400 jest procesorem sześciordzeniowym, pozbawionym obsługi technologii Hyper-Threading zarezerwowanej dla modeli Core i7 oraz wybranych Pentiumów, dlatego wykonuje maksymalnie sześć wątków. Jednostce zwiększono natomiast rozmiar pamięci cache L1 (128 → 192 kB), cache L2 (1024 → 1536 kB) oraz cache L3 (6144 → 9216 kB), mającej niebagatelne znaczenie w grach komputerowych. Dwukanałowy kontroler pamięci obsługuje teraz oficjalnie moduły DDR4-2666, chociaż może współpracować ze znacznie szybszymi zestawami (np.: 3200 MHz jakich używam w testach dodatkowych), warunkiem jest jednak posiadanie płyty głównej z chipsetem Intel Z370. Bazowe taktowanie Core i5-8400 wynosi 2800 MHz, niemniej faktyczna wartość jest znacznie wyższa, sięgając 3800 MHz podczas obciążenia wszystkich rdzeni. Maksymalne przyspieszenie Turbo Boost dla pojedynczego wątku to natomiast 4000 MHz. Trzeba przyznać, że taktowania są naprawdę wysokie, stawiając Core i5-8400 tylko nieznacznie za Core i5-8600K, osiągającego przeważnie 4100 MHz. Żeby było ciekawiej - nawet na chłodzeniu fabrycznym, pomimo wysokich temperatur, Core i5-8400 utrzymuje takie częstotliwości. PS: Na płycie głównej jakiej używam, nie udało się wymusić maksymalnego Turbo Boost dla wszystkich rdzeni (4000 MHz). Procesor pomimo ustawiania mnożnika x40 dla każdego rdzenia, synchronizacji oraz Enhanced Turbo, nadal pracował przy 3800 MHz.
